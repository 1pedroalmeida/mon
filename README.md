## Installation

```sh
git clone https://github.com/1pedroalmeida/mon $HOME/appdata/local/mon --depth 1
cd $env:LOCALAPPDATA/mon
deno compile -A mod.ts
# Add $HOME/appdata/local/mon to PATH
```
